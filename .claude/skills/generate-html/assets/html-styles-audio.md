# HTML Inline Style Reference

All styles must be inline (`style=""` attributes) for email client compatibility — no `<style>` block, no CSS classes, no CSS variables.

## Markdown → HTML mapping

- `# heading` → `<h1 style="font-size:28px; font-weight:700; margin:0 0 4px 0; color:#1a1a2e; border-bottom:3px solid #364fc7; padding-bottom:8px;">`
- `## heading` → `<h2 style="font-size:20px; font-weight:600; margin:40px 0 16px 0; color:#1a1a2e; text-transform:uppercase; letter-spacing:0.05em;">`
- `### [title](url)` → `<h3 style="font-size:17px; font-weight:600; margin:0 0 8px 0;"><a href="url" style="color:#1a1a2e; text-decoration:none; border-bottom:2px solid #364fc7;">title</a></h3>`
- Overview paragraph → `<div style="background-color:#edf2ff; border-left:4px solid #364fc7; padding:16px 20px; margin:24px 0; font-size:15px; color:#495057;">...</div>`
- "What happened" / "What to watch" blocks → `<div style="background-color:#edf2ff; border-left:4px solid #364fc7; padding:16px 20px; margin:24px 0 12px 0; font-size:15px; color:#495057;"><strong style="color:#1a1a2e;">What happened:</strong> ...</div>` (same style for both blocks)
- Relevant item card (from `###` through key moments) → `<div style="background-color:#f8f9fa; border:1px solid #dee2e6; padding:20px 24px; margin-bottom:20px;">...</div>`
- `**Source**: ... | **Guest(s)**: ... | **Published**: ...` metadata → pill spans:
  ```html
  <div style="margin-bottom:12px; font-size:13px;">
    <span style="display:inline-block; padding:3px 10px; margin:0 4px 4px 0; font-weight:500; background-color:#d0ebff; color:#1864ab;">Source: <a href="..." style="color:#1971c2; text-decoration:none;">name</a></span>
    <span style="display:inline-block; padding:3px 10px; margin:0 4px 4px 0; font-weight:500; background-color:#fff3bf; color:#e67700;">Guest(s): name</span>
    <span style="display:inline-block; padding:3px 10px; margin:0 4px 4px 0; font-weight:500; background-color:#e6fcf5; color:#087f5b;">Published: date</span>
  </div>
  ```
- Key moments list → `<ul style="list-style:none; padding:0; margin:0;">` with `<li style="padding:6px 0 6px 20px; font-size:14px; color:#495057; border-left:2px solid #dee2e6; margin-left:4px;">...</li>`
- Key moment timestamp links → `<a href="..." style="color:#364fc7; font-weight:600; font-family:Courier New, monospace; font-size:13px; text-decoration:none;">`
- Tables → `<table style="width:100%; border-collapse:collapse; font-size:14px; margin-bottom:16px;">` with:
  - `<th style="background-color:#f1f3f5; text-align:left; padding:8px 12px; font-weight:600; color:#495057; border:1px solid #dee2e6;">`
  - `<td style="padding:8px 12px; color:#495057; border:1px solid #dee2e6;">`
- No new content list → inline spans: `<span style="display:inline-block; padding:4px 11px; margin:0 4px 4px 0; background-color:#e9ecef; color:#495057; font-size:13px;">...</span>`
- Inline due diligence flags (italic parentheticals within key moments) → `<em style="color:#e67700; font-size:13px;">`
- Links → `<a href="url" style="color:#1971c2; text-decoration:none;">text</a>`
- `**bold**` → `<strong style="color:#1a1a2e;">`, `*italic*` → `<em>`
- Footer → `<div style="margin-top:48px; padding-top:16px; border-top:1px solid #dee2e6; font-size:12px; color:#868e96; text-align:center;">Generated by Claude Stalk</div>`
